to use a converter
- make a converter folder in any folder you'd like to add a CustomPaintings folder to (to be able to send profile codes, the config folder is recommended)
- create a folder called ImagesToConvert in the converter folder
- put all images you want to convert in this folder
- launch the converter of choise
- done

images will automatically be moved to the CustomImages folder and renamed to numbers from 1 up
images in the ImagesToConvert folder will be deleted
